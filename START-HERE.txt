  Q: Do I really need to restart my computer?
  A: YES! If Node.js was just installed, you MUST restart your computer,
     then run install.bat AGAIN to complete the Etherith installation.

  Q: Do I need to run install.bat twice?
  A: Only if Node.js wasn't already on your computer. The installer
     will tell you if you need to restart and run it again.

  Q: Where do I type commands?
  A: In the Command Prompt (the black window that opens when you
     press Windows key + type "cmd" + Enter)

  Q: Is this safe?
  A: Yes! Etherith is completely safe and free.

  Q: Are my files private?
  A: NO - files uploaded to IPFS are PUBLIC. Don't upload sensitive
     information like Social Security numbers, passwords, etc.

  Q: How much does it cost?
  A: FREE! No fees, no subscriptions, no limits.

  Q: Can I delete files after uploading?
  A: NO - files on IPFS are permanent. This ensures your family
     memories are preserved forever!
